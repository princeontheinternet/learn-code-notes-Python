1. Text Editors 
   
        ● Eg: Sublime Text, Notepad++

2. Full IDEs
   
        ● Eg: Pycharm, intellij, Spyder

3. Notebook Environments
   
        ● Eg: Jupiter Notebook. They support markdown notes.
        ● ipynb - **I**nteractive **PY**thon **N**ote**B**ook is the extension for Jupyter Notebooks.
